```python
# complex number
x = 1 + 2j

# normal division
10 / 3

# quotient
10 // 3

# remainder
10 % 3

# exponent
10 ** 2
```

```python
# augmented assignment operator
x += 3
```

### Functions

```python
# rounds to nearest integer
round(2.9) # to 3

abs(-2.9) # to 2.9
```

### `math` module

```python
import math

math.ceil(2.2) # returns 3
```
