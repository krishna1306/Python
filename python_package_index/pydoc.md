# `pydoc` - To access documentation of a package / module

```bash
pydoc3 math
```

### Create a HTML file for documentation

```bash
# "-w" means write
pydoc3 -w some_module_name
```

### Access documentation in a browser

```bash
# Starts a web server on the port "1234" (indicated by '-p')
pydoc3 -p 1234
```
